# Simple Calculator in C

This is a simple, command-line calculator program built using the C programming language. It performs basic arithmetic operations based on user input.

## Features
* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)

## How to Run the Program
1. Open the project in Code::Blocks (or any C compiler/IDE).
2. Open the main.c file.
3. Press F9 (or click Build and Run).
4. Enter an operator when prompted (e.g., +, -, *, /).
5. Enter two numeric operands to see the result.
